Comment integrer vos fichiers GEO

Vous avez recu vos fichiers d'optimisation. Voici comment les mettre en ligne sur votre site, etape par etape, que vous soyez sur WordPress, Webflow, Shopify ou un hebergement classique.

4 fichiers, 4 emplacements

robots.txt
Racine du site
votresite.com/robots.txt
llms.txt
Racine du site
votresite.com/llms.txt
Schema.org JSON-LD
Dans le <head> de la page d'accueil
Page FAQ
Nouvelle page sur votre site
votresite.com/faq

1. Installer le fichier robots.txt

Le fichier robots.txt doit etre place a la racine de votre site web. C'est le premier fichier que les crawlers IA consultent.

Connectez-vous a votre hebergeur via un client FTP (FileZilla, Cyberduck). Naviguez vers le dossier racine de votre site (souvent public_html/ ou www/). Uploadez le fichier robots.txt a la racine. S'il existe deja, remplacez-le. Verifiez : ouvrez votresite.com/robots.txt dans votre navigateur.

Deux options : (1) Si vous utilisez Yoast SEO : Yoast > Outils > Editeur de fichiers > robots.txt. Collez le contenu et enregistrez. (2) Si vous utilisez RankMath : RankMath > Reglages generaux > Editer le robots.txt. (3) Sinon, uploadez le fichier via FTP dans le dossier racine de WordPress (meme niveau que wp-config.php).

Dans les parametres du projet : Project Settings > SEO > Robots.txt. Collez le contenu genere dans le champ prevu et publiez le site.

Shopify ne permet pas de modifier directement le robots.txt depuis l'interface. Allez dans Theme > Edit Code > Templates > robots.txt.liquid. Modifiez le contenu selon les regles generees.

2. Installer le fichier llms.txt

Le fichier llms.txt doit egalement etre place a la racine de votre site. C'est le fichier que les IA lisent en priorite pour comprendre votre activite.

Meme procedure que robots.txt. Uploadez llms.txt a la racine via FTP. Verifiez : votresite.com/llms.txt doit afficher le contenu Markdown.

Uploadez le fichier via FTP dans le dossier racine de WordPress (meme niveau que wp-config.php). Attention : ne l'uploadez pas dans /wp-content/ mais bien a la racine. Verifiez que l'URL votresite.com/llms.txt fonctionne.

Webflow ne permet pas de placer des fichiers a la racine. Solution : uploadez le fichier sur un service d'hebergement statique (ex: Netlify Drop, GitHub Pages) sur un sous-domaine, ou utilisez une regle de redirection dans les parametres Webflow.

Shopify ne permet pas de placer des fichiers arbitraires a la racine. Creez une page dediee a l'adresse /pages/llms-txt avec le contenu en texte brut, ou utilisez un service tiers avec redirection.

3. Integrer le balisage Schema.org

Le code Schema.org JSON-LD doit etre insere dans la balise <head> de votre page d'accueil. C'est un bloc <script> invisible pour les visiteurs mais lu par Google et les IA.

Ouvrez le fichier HTML de votre page d'accueil (souvent index.html). Collez le bloc <script type="application/ld+json">...</script> juste avant la balise </head>. Enregistrez et uploadez le fichier.

Plugin recommande : WPCode (Insert Headers and Footers). Installez le plugin, allez dans WPCode > En-tete et pied de page > Header. Collez le bloc script. Sinon avec RankMath : RankMath > Schema (sur la page d'accueil). Ou avec Yoast : utilisez un plugin complementaire pour le JSON-LD personnalise.

Project Settings > Custom Code > Head Code. Collez le bloc <script type="application/ld+json"> dans le champ "Head Code". Publiez le site.

Theme > Edit Code > Layout > theme.liquid. Collez le bloc script juste avant la balise </head>. Enregistrez.

4. Creer la page FAQ

La page FAQ est un fichier HTML complet a heberger comme nouvelle page sur votre site. Elle contient deja le balisage FAQPage Schema.org.

Creez un fichier faq.html dans le dossier de votre site. Collez le contenu genere. Ajoutez un lien vers cette page dans votre navigation. Verifiez : votresite.com/faq doit afficher la page.

Pages > Ajouter. Passez en editeur HTML/Code (pas visuel). Collez le contenu HTML genere. Publiez la page. Ajoutez-la dans votre menu (Apparence > Menus). Le balisage Schema.org FAQPage est deja inclus dans le HTML.

Pages > Ajouter. Ajoutez un composant "Embed" dans la page. Collez le contenu HTML. Publiez le site. Ajoutez la page a votre navigation.

Online Store > Pages > Add page. Passez en mode HTML dans l'editeur. Collez le contenu. Publiez. Ajoutez la page a votre menu de navigation (Navigation > Main menu).

Confiez l'integration a votre prestataire

Si vous preferez ne pas manipuler les fichiers vous-meme, transmettez ce message a votre developpeur ou agence web :

Bonjour, Je souhaite integrer 4 fichiers d'optimisation GEO sur mon site [URL DU SITE]. Voici les fichiers a installer : 1. robots.txt - a placer a la racine du site (remplace l'existant si necessaire) 2. llms.txt - a placer a la racine du site (nouveau fichier) 3. Schema.org JSON-LD - a coller dans le <head> de la page d'accueil (bloc <script type="application/ld+json">) 4. Page FAQ - a creer comme nouvelle page accessible depuis /faq Les fichiers sont joints a cet email. Merci de confirmer la mise en ligne.
